# Social post templates

Reusable social-post templates for the 2026-07-19 showcase release, the live
autonomous run written up in
`docs/_posts/2026-07-19-a-factory-that-builds-software-one-live-run.md`.

- `reddit-post.md` — Reddit submission (title options, body, and a
  comments-section FAQ). Leads with the honesty gate, not the product.
- `linkedin-post.md` — LinkedIn post (first-person-plural, 150-250 words, hook +
  body + hashtag block).

These are drafts to adapt before posting, not final copy. Match each community's
rules and tone, keep the claims to what the blog post supports, and do not add
emojis.
