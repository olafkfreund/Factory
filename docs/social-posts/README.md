# Social post templates

Ready-to-adapt announcement copy for the Factory program. Each file is a
template: swap the bracketed placeholders, trim to the platform's norms, and
post. Plain text only, no emojis, matching the program's writing style.

## Files

- `linkedin-daily-driver.md` — LinkedIn milestone announcement (professional, concise).
- `reddit-daily-driver.md` — Reddit technical post (honest, community-oriented; r/programming, r/devops, r/LocalLLaMA).
- `linkedin-feature-highlights.md` — shorter LinkedIn variant focused on one feature at a time.

## Placeholders

- `[BLOG_URL]` — link to the matching post under `/blog/`.
- `[REPO_URL]` — the public repository or landing page.
- `[METRIC]` — a concrete number when you have one (token reduction, test count, run time).

## House rules

- No emojis, no hype adjectives without a number behind them.
- Every claim maps to something shipped and verifiable.
- Lead with the concrete result, not the technology.
- Link to the blog post for anyone who wants the detail.
